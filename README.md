PremierStats

Aplikacja mobilna została zaprojektowana w celu umożliwienia użytkownikom wygodnego dostępu do informacji związanych z rozgrywkami Premier League. Jej głównym zadaniem jest prezentowanie aktualnych danych dotyczących tabeli ligowej, wyników spotkań, bramek zawodników orazstatystyk meczowych.
Interfejs aplikacji jest przejrzysty i intuicyjny, co pozwala użytkownikowi szybko odnaleźć interesujące informacje. Nawigacja odbywa się za pomocą menu oraz dolnego paska nawigacyjnego.
Aplikacja składa się z następujących modułów:
-tabela ligowa,
-mecze i wyniki,
-statystyki meczu,
-bramki,
-wiadomości,
-ustawienia użytkownika.
